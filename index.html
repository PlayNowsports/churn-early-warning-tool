How to Use This Tool

1. Review or Edit Rules

The tool ships with 12 default rules covering both structured signals (Health field, Utilization, Age, Stage) and unstructured patterns (competitor mentions, champion turnover, budget freezes, integration gaps). Edit them, add your own, or reset to defaults. Rules persist in browser storage.

2. Upload Your Pipeline Export

Export renewal opportunities from your CRM as .xlsx or .csv. Map your columns to the standard fields (Account Name, ARR, Health, CSM Notes, etc.). Only the mapped columns are used.

3. Run the Analysis

Structured rules run instantly, in-browser, deterministically. Unstructured rules require an Anthropic API key. Accounts are batched (default 25 per call) to keep costs and latency low.

4. Read the Results

Accounts are tiered Critical / Elevated / Watch / Healthy based on total score. Each account shows which specific rules fired and why — no black-box scoring. Export to CSV for CRM upload or QBR handoff.

Portability

Export your rule set as JSON to move it between machines or version it in git. The tool is a single static HTML file — deployable to GitHub Pages, S3, any CDN, or opened directly from disk.

Privacy

All data processing happens in your browser. The only network call is to api.anthropic.com when unstructured rules are enabled, and that uses your own API key. Nothing is logged or stored on any server.
